# How to Add Your CashApp QR Code

## üì∏ Step 1: Get Your CashApp QR Code

1. Open the **CashApp** on your phone
2. Tap your **profile icon** (top right)
3. Tap **"Cash App Tag"** or **"QR Code"**
4. Take a screenshot or save the QR code image

## üìÅ Step 2: Add QR Code to Repository

### Option A: Add via GitHub (Easiest)

1. Go to your repository: https://github.com/abhishek-donekal/abhishek-donekal
2. Click **"Add file"** ‚Üí **"Upload files"**
3. Drag and drop your QR code image
4. Name it: `cashapp-qr.png`
5. Click **"Commit changes"**

### Option B: Add Locally

1. Save your QR code image as `cashapp-qr.png`
2. Copy it to: `D:\Projects\abhishek-donekal\`
3. Then run:
   ```powershell
   cd D:\Projects\abhishek-donekal
   git add cashapp-qr.png
   git commit -m "Add CashApp QR code"
   git push origin main
   ```

## ‚úÖ Step 3: Verify

After adding the image, your profile README will automatically display the QR code!

## üé® Image Requirements

- **Format:** PNG or JPG
- **Size:** 200x200 to 400x400 pixels works best
- **Name:** `cashapp-qr.png` (must match exactly)
- **Location:** Root of the repository

## üí° Tips

- Make sure the QR code is clear and scannable
- Use a white or light background for best visibility
- Test the QR code before uploading to make sure it works
